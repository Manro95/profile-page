Website

This is a responsive personal portfolio website built with HTML5 and CSS3. It showcases an about section, featured projects, and contact links, and is designed as a clean, single-page layout suitable for deployment on GitHub Pages.

✨ Features

Fixed navigation bar with anchor links

Fullscreen hero/about section

Project showcase using CSS Grid

Hover effects for links and icons

Responsive design for mobile screens

Social and contact links (Email, LinkedIn, GitHub)

🛠️ Technologies Used

HTML5 – semantic structure

CSS3 – Flexbox, Grid, gradients, media queries

Font Awesome – social media icons

📁 Project Structure
├── index.html
├── styles.css
└── README.md

📱 Responsive Design

The layout adapts to smaller screens using a media query:

Navigation and footer text scale down on screens under 600px

Grid layout remains clean and readable on mobile devices

🎨 Customization

You can easily customize:

Colors & gradients in styles.css

Project cards by duplicating .project-card blocks

Social links in the footer section
